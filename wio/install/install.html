Wio is a compiled command line tool and hence is available for pretty much all popular platforms. If a platform is not supported, let the developers know and they can publish a binary for that platform. Current following platforms are supported:

  • Linux
    • Wio build Tools
    • Platform toolchain
      • Atmel AVR toolchain
  • MacOS
    • Homebrew
    • NPM
    • Wio build Tools
    • Platform toolchain
      • Atmel AVR toolchain
  • Windows
    • Scoop
    • NPM
    • Wio build Tools
    • Platform toolchain
      • Atmel AVR toolchain
